# Vue 前端测试指南

## 快速开始

### 1. 启动后端服务

```bash
# 在 D:\LaodengWebProject 目录
python app.py
```

后端应运行在 `http://localhost:5000`

### 2. 启动前端服务

```bash
# 在 D:\vue_project\web_game_vue 目录
npm run dev
```

前端应运行在 `http://localhost:5173`

### 3. 打开浏览器

访问 `http://localhost:5173`

## 调试工具

### Vue DevTools

1. 安装Vue DevTools浏览器扩展
2. 打开浏览器开发者工具（F12）
3. 切换到"Vue"标签页
4. 可以查看：
   - 组件树
   - Store状态
   - 事件日志

### 浏览器控制台

打开开发者工具（F12），查看：
- **Console**：JavaScript错误和日志
- **Network**：API请求和WebSocket消息
- **Application**：LocalStorage和SessionStorage

## 常见问题

### 连接失败

1. 检查后端是否运行
2. 检查端口是否正确
3. 查看浏览器控制台错误信息

### 静态资源404

1. 检查Vite代理配置
2. 确认后端static目录存在
3. 检查图片路径

### 数据不更新

1. 检查WebSocket连接
2. 查看Store状态
3. 检查事件监听
